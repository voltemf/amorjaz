# 💝 Bear Love Website

A romantic interactive website built with HTML, CSS, and JavaScript featuring a lockscreen, swipe navigation, and gallery for Heidy.

## 🌟 Features

- **🔒 Heart Lockscreen**: Password-protected entry with heart profile
- **🐻 Interactive Bear**: Clickable bear with zoom functionality revealing love message
- **💕 Romantic Page**: Love message with heart decorations and bear background
- **📸 Our Gallery**: Photo gallery with clickable images showing Spanish love messages
- **📱 Touch Navigation**: Swipe left/right to navigate between pages
- **💖 Clickable Heart**: Top-left heart showing dedication message
- **✨ Smooth Animations**: CSS animations throughout for beautiful interactions

## 🚀 Quick Start

1. **Clone or Download** this repository
2. **Upload files** to your GitHub repository
3. **Enable GitHub Pages** in repository settings
4. **Visit your site** at `https://yourusername.github.io/repository-name`

## 📁 File Structure

```
├── index.html          # Main HTML file
├── style.css          # All styling and animations

└── README.md          # This file
```

## 💻 Technical Details

- **Pure HTML/CSS/JS**: No frameworks or build tools required
- **Responsive Design**: Works on desktop and mobile
- **Touch Support**: Swipe navigation for mobile devices
- **Local Storage**: Saves editable text between sessions
- **Image Fallbacks**: Graceful handling of failed image loads
- **Smooth Animations**: CSS transitions and keyframe animations

## 🌐 GitHub Pages Deployment

1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click "Save"
6. Your site will be available at the provided URL

## 📝 Customization

- **Images**: Replace URLs in `script.js` gallery array
- **Colors**: Modify CSS custom properties in `style.css`
- **Text**: Update content directly in `index.html`
- **Password**: Change password check in `script.js`
